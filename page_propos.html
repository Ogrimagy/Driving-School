Vos Questions

• Apprentissage en Ligne?

Nous offrons la possibilité d'étudier le code de la route à travers nos rubriques, ces derniers incluent aussi les panneaux de signalisation. Alternativement, vous pouvez choisir d'assisster à nos cours en présentiel si vous le souhaiter.


• Horaire personalisé?

En effet, avec les cours disponible en ligne, vous pouvez etudiez où et quand vous voulez! Un test QCM est également fournis afin de déterminer si vous êtes apte à passez le Code en présentiel.


• Un test sans déplacement?

Nous méttons à votre disposition un test QCM qui servira à juger si vous êtes apte à passer l'examen de Code.


• Comment se passe le suivi?

Tous vos résultats sont enregisté et vous sont fournis en ligne dès qu'ils sont prés. En temps de besoin, vous pouvez aussi nous appeler directement pour une quelconque clarification.
